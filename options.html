Search websites

Add, edit, delete, enable, and reorder search links. Use {term} where the selected term belongs. Optional placeholders: {sourceLang}, {targetLang}, {wordReferencePair}, {lingueePair}.

    Hotkeys can be changed at chrome://extensions/shortcuts. Defaults: lookup selected term = Ctrl/Command+Shift+L; save selected sentence = Ctrl/Command+Shift+S.

    Hover-capture hotkey

    This one is press-and-hold, so it's set here instead of chrome://extensions/shortcuts (which only supports single key presses). Release the combo or press Esc to cancel a capture. Default: Ctrl+Shift+K.

    Cloze behavior

    Storage

    While signed out, saved vocabulary is local to Chrome. Export/import keeps a readable backup file that feels more tangible than hidden browser storage.

    Sync

    Sign in with the account you created in the Firebase console to keep vocab in sync with the phone web app. Signing in moves any vocab saved in Chrome to your account and clears it from Chrome's local storage — from then on your vocab lives only in the cloud, so you'll need a connection to load or save it. Signing in here also lets the Vocab Toolkit tab (Review page) use the same account.
